## 纯属兴趣爱好
1. 这是一个进程代理项目，使用Netfilter驱动拦截进程流量，将其代理
2. 因为很多软件都不会内置代理功能，且不会去读取系统代理设置，那么进程劫持代理将会很有用 
3. 这是对(<a href="https://github.com/netchx/netch" target="_blank">netch</a>)，项目的一个简化，只保留了进程代理模式，及只保留了socks5代理协议
